# 🛒 Amazon Clone (Flutter)

A responsive Amazon Clone UI built using Flutter.  
This project is designed to replicate a modern e-commerce experience similar to Amazon.

---

## ✨ Features

- 🏠 Home screen with product listings  
- 🔍 Search UI (basic or dynamic depending on implementation)  
- 🛍️ Product detail page  
- 🛒 Add to Cart functionality  
- ❤️ Wishlist UI (optional)  
- 👤 Profile screen UI  
- 🎨 Clean Amazon-inspired UI design  
- 📱 Responsive for mobile, web, and desktop
